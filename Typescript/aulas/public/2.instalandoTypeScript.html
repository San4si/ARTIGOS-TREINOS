Instalando TypeScript

Entre no site do TypeScript em typescriptlan.org/download

Para instalar o TypeScript precisara do Node instalado para usar o gerenciador de pacotes NPM.

Entre no site do node em nodejs.org/en/

baixe a versão LTS recomendada.

Apos instalado para testar abra o prompt de comando e digite:
npm --version
aparecera o versão do NPM, onde indica que o node esta instalado.

O proximo passo é instalar o TypeScript no proprio prompt rode o comando:
npm install -g typescript
mostrara a versão.

Para testar se esta funcionando rode o comando: tsc --version
mostrara a versão

PRONTO O NODE E O TYPESCRIPT ESTÃO INSTALADOS PRONTOS PARA TRABALHAR

! Deverá colocar no arquivo HTML o script javascript e não o ts, mesmo não existindo o arquivo js,
o ts criara quando for convertido com comando abaixo.

Para executar nosso projeto sera necessario converter arquivo .ts para js
Com o seguinte comando no terminal: ts script.ts
Após a execução aparecera o arquivo convertido para javascript.

O arquivo typescript não roda no navegador, serve apenas para desenvolver.

Os arquivos devem ser separado em os que vao a publico e desenvolvimento.
geralmente separamos em pastas com o nome 'public' e 'src'.
onde os arquivos js e html irao para public e src o codigo ts.

Ao separar em pastas quando converter o arquivo ts que esta dentro da pasta 'src' deve ser gerado o js
dentro da pasta ' public '. No terminal deve digitar o seguinte comando: tsc src/script.ts --outDir public
Com o comando' --outDir public ' ira colocar o arquivo js convertido dentro da pasta ' public '.
