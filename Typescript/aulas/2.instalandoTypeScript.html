Instalando TypeScript

Entre no site do TypeScript em typescriptlan.org/download

Para instalar o TypeScript precisara do Node instalado para usar o gerenciador de pacotes NPM.

Entre no site do node em nodejs.org/en/

baixe a versão LTS recomendada.

Apos instalado para testar abra o prompt de comando e digite:
npm --version
aparecera o versão do NPM, onde indica que o node esta instalado.

O proximo passo é instalar o TypeScript no proprio prompt rode o comando:
npm install -g typescript
mostrara a versão.

Para testar se esta funcionando rode o comando: tsc --version
mostrara a versão
