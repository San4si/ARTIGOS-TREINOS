Scroll em barra de rolagem em elemento ou tela inteira, manipulações para encontrar scroll ou criar scroll suave
Como saber onde esta o scroll: Tela inteira, scroll da propria tela usamos o: windown
se for de um elemento especifico selecionamos o elemento: document.querySelector('.texto').scrollTop para pegar a posição do scroll vertical document.querySelector('.texto').scrollLeft pega o scroll horizontal No caso da tela inteira para pegar a informação: window.scrollY para vertical window.scrollX para vertical Comando para alterar o scroll document.querySelector('.texto').scrollTo(0,0)
Comando para scroll suave document.querySelector('.texto').scrollTo(){window.scrollTo({top: 0, left: 0, behavior: 'smooth'})};
