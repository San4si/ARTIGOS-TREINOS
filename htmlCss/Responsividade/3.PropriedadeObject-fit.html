Imagens responsiva

A propriedade Object-fit trata da adaptalidade de uma imagem no css
No css em um tamanho fixo via css podemos corrigir tamanhos quadrados
com a propriedade object-fit onde tem varias opções:
Object-fit:fill = Opção padrão onde sera redimensionada independente da proporsão
de forma a mostrar inteira na area dispovel mesmo sendo distorcida.
Object-fit:contain = diminui a imagem, mantendo a proporção de forma que a
imagem inteira aparece no espaço disponivel sem distorcer.
Object-fit:cover = è um dos mais utilizados, mantem a proporsão da imagem,
porem corta de forma a preenxer todo o espaço disponivel, e pra não distorcer a imagem é cortada no meio no centro da imagem.
Object-fit:none = Corta a imagem sem redimensionar
Object-fit:scale-dow = diminui pra menor versao que a imagem pode ter, similar ao contain
